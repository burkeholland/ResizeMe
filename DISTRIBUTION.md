# How to Install ResizeMe

## Recommended Method (Simple Installer)

1. **Unzip** the `ResizeMe_1.1.2.0_Installer.zip` file.
2. Open the folder.
3. Double-click **`Install ResizeMe.bat`**.

This will:
*   Install the app to your local programs folder.
*   Create a shortcut on your Desktop.
*   Create a shortcut in your Start Menu.

**No extra downloads required.** (Includes everything needed to run).

## Alternative: MSIX Installer
(Use this if you want a managed install/uninstall experience)

1. Unzip `ResizeMe_1.1.2.0_Test_Build.zip`.
2. Run `install.bat` as Administrator.
